# Install and update

The planned public source is `https://github.com/metin-korkmaz/lunaglow-rp6`.

## Direct QR installation

Scan `docs/lunaglow-latest-qr.png`. It opens the stable release asset:

`https://github.com/metin-korkmaz/lunaglow-rp6/releases/latest/download/lunaglow-rp6.apk`

Android requires two legitimate confirmations:

1. Allow the browser or Obtainium to install unknown apps once.
2. Confirm each installation or update in Android's package installer.

The QR code cannot and does not bypass Play Protect or Android's installer.

## Obtainium updates

1. Install Obtainium from its official GitHub or F-Droid source.
2. Add `https://github.com/metin-korkmaz/lunaglow-rp6` as an app source.
3. Select the `lunaglow-rp6.apk` release asset.
4. Allow Obtainium to install unknown apps when Android asks.

Obtainium will detect later GitHub Releases. Updates must have a higher
`versionCode` and must be signed with the same LunaGlow release key.
